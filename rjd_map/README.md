# Дислокация вагонов РЖД — Веб-карта

Готовый к деплою PHP+JS проект для отображения вагонов на карте.

## Структура
```
wagon_map/
├── index.php   — главный файл приложения
└── data.json   — данные из справок АСОУП (1 409 вагонов, 45 станций)
```

## Стек (всё бесплатное)
| Компонент | Сервис | Лицензия |
|-----------|--------|----------|
| Карта | **Leaflet.js** + **CartoDB Light** тайлы | MIT / бесплатно |
| Кластеризация | **Leaflet.MarkerCluster** | MIT |
| Шрифты | **Google Fonts** (Inter + JetBrains Mono) | бесплатно |
| Бэкенд | PHP 7+ (просто отдаёт JSON в HTML) | — |

## Деплой

### Вариант 1 — Любой shared-хостинг (Timeweb, REG.RU и т.д.)
1. Загрузить `index.php` и `data.json` в корень сайта или папку
2. Открыть в браузере

### Вариант 2 — Локально (без сервера)
Заменить в `index.php` PHP-строку:
```php
$wagon_data = file_exists($data_file) ? file_get_contents($data_file) : '[]';
```
...на прямое встраивание данных (для статики). Либо запустить:
```bash
php -S localhost:8080
```

### Вариант 3 — Nginx / Apache
Положить файлы в `/var/www/html/` — работает «из коробки».

## Функциональность
- **Карта** с кластеризацией маркеров (Leaflet)
- **Боковая панель** со списком станций и счётчиками
- **Фильтры**: Все / Гружёные / Порожние / Простой >5 дней
- **Поиск** по станции и по номеру вагона
- **Детальная панель** по каждой станции с полным списком вагонов:
  - Номер, тип, груз, состояние (РП / ТР / ДР)
  - Станция назначения
  - Дней без движения

## Обновление данных
Заменить `data.json` новым файлом, сгенерированным из АСОУП-справки.
Скрипт генерации — `generate_data.py` (если нужен, запросите).
